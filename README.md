# AI Health Triage Assistant (BYOP Capstone)

## 📌 Project Overview
The **AI Health Triage Assistant** is a rule-based **Intelligent Agent** designed to solve the problem of hospital overcrowding. By automating the initial screening process, the agent identifies patients requiring immediate life-saving intervention versus those who can be treated with home care.

## 🧠 AI Concepts Applied
This project directly implements concepts from the course modules:
* **Module 1: Intelligent Agents** - Classified as a **Simple Reflex Agent**.
* **PEAS Framework Analysis**:
    * **Performance**: Speed of classification and accuracy of triage priority.
    * **Environment**: Medical reception desk/Patient symptom data.
    * **Actuators**: Digital display/Instructional output for medical staff.
    * **Sensors**: Human-interactive text input (Perception).
* **Module 2: Problem Solving** - Uses a **Decision Tree** logic to navigate through symptoms.

## 🛠️ How to Use
1. **Download**: Save `triage_bot.py` to your computer.
2. **Execute**: Open a terminal or Python IDLE and run `python triage_bot.py`.
3. **Interact**: Follow the on-screen prompts. The agent handles errors if you type something other than 'yes' or 'no'.

## 📁 Repository Structure
- `triage_bot.py`: The core logic of the AI agent.
- `README.md`: Project documentation.
- `.gitignore`: Ensures system files are not tracked.
